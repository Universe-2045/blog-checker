[블로그 체커 열기] (index.html)

1. 카카오톡 블마방 -> 대화 내보내기
2. checker.html 웹에서 열기
3. 챌린지 시작일 설정
4. 체크할 날짜 설정
5. 챌린지 이름 입력
6. 참여자 명단은 기존 명단으로 입력 (추가시 추가입력)
7. 닉네임 별칭 매팅은 카톡방과 블로그명이 다른경우 추가
8. 카카오톡 대화 내보내기 파일을 업로드
9. 체크 실행 버튼 클릭 -> 하단에 카톡 붙여 넣기 메시지 생성
   전체 인원 = 인증 + 미인증, 참여자 현황   
10. 카톡에 붙여넣기용 메시지를 자동 생성 (복사해서 사용)

*** 날짜 설정, 명단 드을 설정 등을 내보내기/불러오기 가능
*** 통계 저장은 안됨
